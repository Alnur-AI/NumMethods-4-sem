Данная программа выполняет интерполяцию Лагранжа на неравномерной сетке

Запускать программу следующей командой: python3 task3.2.py


test.ans - выводные данные по оси У (формируется по формуле интерполяции Лагранжа)
test.dat - выводные данные по оси Х (из этих данных формируется новая интерполяция)
train.ans - вводные данные по оси У (по ним мы строим интерполяцию)
train.dat - вводные данные по оси Х (по ним мы строим интерполяцию)
